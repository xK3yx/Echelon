Notes for any Claude Code session (or other contributor) working in this repo.

- All commits are authored by **xK3yx**.
- **Do not** add `Co-Authored-By:` trailers, AI attribution lines, or generated-with footers to commit messages.

## Commit messages

Plain conventional commits — `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`, `test:`. Short subject, optional body explaining the *why*. No footers other than `BREAKING CHANGE:` when applicable.
